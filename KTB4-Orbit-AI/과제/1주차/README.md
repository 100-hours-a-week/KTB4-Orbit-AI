# KTB4-Orbit-AI
WEEK1 CLI programming

> KTB4 위클리챌린지 1주차  - CLI 기반 배운 용어들 공부 및 퀴즈를 통한 오답노트 시스템

---

## 목표 

CLI환경에서 동작하는 개발 용어 학습 및 복습 프로그램 구현
- note.md 파일 기반의 용어 정리집 연동
- asyncio 를 활용한 10초 제한시간 타이머 및 비동기 입력 처리
- 틀린 문제를 자동으로 분류하여 영속적으로 저장하는 오답노트 기능 구현


## 프로젝트 구조
과제/1주차/
├── Orbit.py          # 메인 실행 파일 (CLI 라우팅 및 퀴즈 엔진)
├── note.md           # 용어 정리집 데이터 (원본 데이터셋)
└── wrong.md          # 틀린 문제 자동 기록 저장소 (오답노트)

## 설치 및 실행
의존성 라이브러리
- 본 프로그램은 표준 내장 라이브러리(asyncio, sys, os, random)만 사용하므로 별도의 설치 과정이 필요하지 않습니다.

```bash
python Orbit.py --quiz
python Orbit.py --wrong
```

python Orbit.py --quiz -> 비동기 퀴즈 실행
python Orbit.py --wrong -> 오답노트 보기

## 주요 기능 
- 퀴즈 및 오답 흐름

10초 이내 정답 입력 -> 정답! 
10초 초과 -> 시간 초과! 
오답 입력 -> 틀렸습니다! 정답은 OOO입니다.

## 비동기 처리 핵심
- asyncio.to_thread : 시간이 오래 걸리는 작업을 별도의 스레드에서 실행
- asyncio.wait_for : 작업의 실행 시간을 제한하고 타임아웃 발생 시 예외 처리

## 데이터 형식
- note.md, wrong.md (텍스트 파일 기반 구조)

```markdown
질문 내용 | 정답 단어
```

- 예시
```markdown
원격 저장소에 있는 레포지토리의 최신 내용을 로컬 저장소로 가져올 때 사용하는 명령어 | git pull
commit 기록을 볼 수 있는 명령어 | git log
파일의 변경 사항을 확인할 때 사용하는 명령어 | git diff
파일의 변경 사항을 GUI 도구로 확인할 때 사용하는 명령어 | git difftool
```

## 회고
파이썬을 거의 처음 접해보며, 하나하나 파이썬 용어들을 차근차근 알아가는 과정에서 어떻게 하면은 좀더
기억에 남고 잘 활용할 수 있을까 고민하다 
CLI로 퀴즈를 풀고 오답노트를 영속적으로 저장하는 프로그램을 구상하게 되었습니다.
아직 파이썬이 익숙하지 않아 AI를 도움을 받아 프로그램을 작성하였지만, 
프로그램을 통하여 파이썬의 문법과 깃&깃허브 명령어 들을 공부하고 익숙해질 수 있도록 노력할 것이며,
AI의 도움을 받아 작성한 코드들 또한 제 것이 될 수 있게 공부를 이어 갈 예정입니다.

처음부터 잘 할 수 없다고 생각합니다. 하지만 꾸준히 열심히 계속해서 공부를 이어나간다면
언젠가는 저도 파이썬을 만질 수 있다 라는 자신감도 가지고 개발자로서 나아갈 수 있다고 생각합니다.
부족하면 시간을 더 써서 공부하고 배우고 노력하면 된다고 생각합니다.
앞으로의 카테부 기간 뿐만 아니라 계속해서 공부하고 배우며, 배운 사람이 아닌 배우는 사람이 되어야 겠다고 생각하며 나아가겠습니다.
